Origins

Keybout started in 2015 as a WebSocket (an advanced communication protocol over the Internet) sample project then evolved to a simple game.

The original implementation was based on: JavaScript, WebSocket, HTML, Bootstrap, Java EE 7, WildFly, Maven

Rewrite

In 2019, to learn Angular and Kotlin, I started a full rewrite of the application.

This second version is based on: Angular, SockJS, HTML, Bootstrap, Spring Boot, Kotlin, Maven
